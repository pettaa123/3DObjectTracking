Detailed Description

Class that shows an image on which the user is able to specify points and that is used by the ManualDetector class.

It is able to load a detector image to visualize the points the user should select. Calling the method DetectPoints() starts the selection process and stores the image coordinates in detected_points.

Parameters
imageimage that is shown.
window_namename of the window in which the image is shown.
detector_image_pathoptional path to a helper image in which the position of reference_points can be marked.
