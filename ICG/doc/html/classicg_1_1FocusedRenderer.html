Detailed Description

Abstract Renderer class that defines a focused renderer that crops and scales the scene to focus on referenced objects with a defined image size.

Objects are only considered if they would be visible in the image defined by the intrinsics. The left upper corner of the crop relative to the image defined by the intrinsics is given by corner_u and corner_v. The relative scale of the image is provided by scale. The visibility of bodies in the rendered image can be checked using IsBodyVisible().

Parameters
image_sizeimage size that is rendered.
referenced_body_ptrsreferenced Body objects on which the scene is focused.
