Detailed Description

Abstract class that provides images to other components and contains intrinsics as well as the pose of the camera relative to the world coordinate frame.

Using the main method UpdateImage(bool synchronized), a new image is obtained. If the flag synchronized is true, a corresponding real-world camera waits until a new image arrives. The class also includes functionality to save images using StartSavingImages() and StopSavingImages().
