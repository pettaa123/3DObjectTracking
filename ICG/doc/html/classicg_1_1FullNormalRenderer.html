Detailed Description

Renderer that extends the full depth renderer class with functionality from NormalRendererCore to render both a depth image and an image where the normal vector of the surface is encoded in the color of each pixel.

Rendering is started using StartRendering(). Images are fetched from the GPU using FetchNormalImage() and FetchDepthImage(). They can then be accessed using the normal_image() and depth_image() getter. Setters and all main methods are thread-safe.
