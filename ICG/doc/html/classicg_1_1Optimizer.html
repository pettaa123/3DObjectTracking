Detailed Description

Class that uses the gradient vectors and Hessian matrices from multiple Modality objects to update the pose of a Body.

With the main method CalculateOptimization(), information from all referenced modalities is used to update the pose of the body referenced by the modalities. It is important that all modalities reference the same body.

Parameters
modality_ptrsdefines the referenced Modality objects that are considered.
