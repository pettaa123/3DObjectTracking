Detailed Description

Abstract class that defines a renderer as a single camera at a defined location.

It creates a rendering based on the geometry stored in the RendererGeometry, the pose of Body objects referenced in the RendererGeometry, and the view of the renderer on the scene defined by intrinsics and a pose. Rendering is started calling the main method StartRendering().

Parameters
renderer_geometry_ptrreferenced RendererGeometry object that defines the geometry that is rendered and provides a GLFW context.
world2camera_posepose of the camera relative to the world frame.
intrinsicsfocal length, principal point, and image size that define the view of the renderer on the scene.
camera_ptroptional parameter that references a Camera object. Executing the SetUp() method writes the world2camera_pose and intrinsics parameters from the camera to the renderer.
z_minminimum z-distance that is rendered.
z_maxmaximum z-distance that is rendered.
